OpenAero manual - General

OpenAero is sequence design software that runs inside your web browser. This makes it available on practically any computer, tablet and even smartphone. However, for the best experience the Google Chrome browser is recommended.
To have maximum space available for using the software it is advisable to use the full-screen mode of the browser by pressing F11 on your keyboard.
The latest version of OpenAero can be used on openaero.net. If you want to download the software for use when you are not online, please refer to Installation.

Creating sequences

Sequences can be created in two ways:

  • By writing them in code in the sequence line
  • By selecting and editing figures with the Figure editor
The desciptions below refer to using the figure editor. For the correct codes for using the sequence line, please refer to OpenAero language.

Beginning a sequence

Start by filling out the Sequence info. The Rules, Category and Program information will be used for determining sequence validity.
In the Sequence menu you can choose to start the sequence downwind or crossbox. This can always be changed at a later stage.

Adding a figure

Go to the Figure editor tab and click on the Click to add figure box. A figure chooser will now appear from which you can choose a group (e.g. 1.3 Three Lines) and a specific base figure.
When you select a figure this will become active and shown in magenta in the sequence area. It will also be shown in the figure area where you can:

  • Switch X-axis or Y-axis exit direction (for some figures)
  • Delete the figure
  • Zoom the figure
  • Move the figure to a different position
  • Extend or shorten the entry and exit lines
  • Add/delete/change rolls
  • Extend or shorten lines before and after rolls
When you are done editing the figure you can click anywhere in the sequence area to deactivate the figure and add a new figure at the end.

Selecting and moving a figure

To select a figure, click on the start or select it in the sequence line. The figure is now magenta and can be edited with the figure editor.
For repositioning, it can be dragged anywhere by holding down the mouse button after clicking the figure start.

Keeping the figures separate

When figures are overlapping they can be dragged to a new position. It is also possible to use Separate figures from the Sequence menu to separate all figures.

Opening, saving and printing sequences

Opening sequences

To open an existing sequence, choose Open sequence in the File menu. You will be asked to choose a file. Sequence files always end in .seq and can be created either with OLAN or OpenAero.

Saving sequences

To save the current sequence as a .seq file, choose Save sequence in the File menu. Your computer will then download the data and, depending on the browser, ask for a file name to save the sequence as. Always ensure to give the sequence a file name ending in .seq! These files can only be opened by OpenAero and not by OLAN.

Saving the current form (A, B or C) as an image

To save the current form as an image, choose Save as image in the File menu. You will then get the option, depending on your browser, to save the image or view it immediately.

Saving or printing all forms

To save or print all forms, choose Print forms in the File menu. If you want to print the rest is self explanatory. If you want to save the forms as a PDF file you can choose to 'print to PDF' on Mac OSX or Linux. On Windows it is possible to have the same option by installing software that does this, such as PDFcreator.
